Tile Lettering

Created for a project in my BFA studio class, spring 2024. My professor, Alexander Landerman, tasked students with creating a non-digital work relating to graphic design. A handcraft. I decided to create a lettering design using hexagonal tiles. I mapped out the tiles to my design, adhered them to a cement baseboard, grouted, and created a custom wood frame for the mat, which is currently functioning as the welcome mat to my house.
